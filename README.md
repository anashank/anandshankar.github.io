# anandshankar.github.io

Simple static personal website for Anand Shankar.

Pages included:
- `index.html` — homepage
- `about.html` — about and skills
- `projects/index.html` — projects listing
- `resume.html` — resume page (add `assets/resume.pdf` to enable download)
- `contact.html` — contact form (Formspree placeholder)

To preview locally open `index.html` in a browser or run a simple HTTP server:

```bash
python3 -m http.server
```

Then open `http://localhost:8000`.
